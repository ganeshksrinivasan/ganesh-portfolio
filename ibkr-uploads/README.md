# IBKR Uploads

Drop your IBKR activity statement CSV here to update the dashboard automatically.

## How to update your dashboard

1. Download your activity statement from IBKR (Reports → Activity → Statements → CSV)
2. Delete the old CSV in this folder
3. Upload the new CSV here
4. GitHub automatically reads it and updates your dashboard within ~1 minute
5. Vercel picks up the change and rebuilds the site

That's it! No manual editing required.
